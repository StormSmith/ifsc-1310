    • Thoughts on Progressive Enhancement

    Progressive enhancement is basically a mindset people in web design use that lets them concentrate on building a website while being able to find problems they may run into while making the website. It's kind of like a step by step list, first you start with the content of the website(html), then you work on the way the website looks (css), then finally you add the javascript to it.
